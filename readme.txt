
#Decision Tree Learning Algorithm for Classification.

This java package contains source files for codes and dataset itself

It contains .jar file in the root folder which is our executable file.
This jar file takes argument which are
i) option to choose entropy (1) or misclassification(2)
ii)chi-square value

Note that parameters are mentioned in the order they need to pass to the program.

To run this executable file go to the command line and type

java -jar 'file_name.jar' 'chi' 'xed'

you will see the output with different accuracy rate printed.
